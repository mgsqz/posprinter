	# ESC/POS RAW Server
	一个轻量级的 ESC/POS 小票打印数据转发与解析服务。它可以接收来自收银系统或 POS 软件的打印数据，将其保存备份，并在网页上实时预览小票内容。同时支持通过 Webhook 触发本地 USB 打印机直接打印原始数据。
	## 功能特性
	- **RAW 数据接收**：监听 TCP 9100 端口，完美接管 POS 软件的打印请求。
	- **实时 Web 预览**：内置响应式网页，将二进制 RAW 数据实时解析为可视化的 HTML 小票，支持中文字体放大、加粗、居中/居右对齐、二维码等指令。
	- **物理尺寸还原**：网页预览采用 58mm (32ch) 宽度等比缩放渲染，随浏览器窗口自适应放大，如同看真实小票。
	- **本地打印机转发**：提供 `/print` Webhook 接口，可将最新接收到的 RAW 数据直接发送给本地连接的 USB 小票打印机（如 `/dev/usb/lp0`）。
	- **状态拦截**：智能处理 POS 软件的联机状态探测指令（如 DLE EOT, GS r），确保软件不会卡在“打印机未连接”状态。
	- **静默运行**：默认全程静默运行，不产生任何日志输出，适合后台无人值守部署。
